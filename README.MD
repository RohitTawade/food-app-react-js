# To Check the code version

```
node --version
```

# To initailize the Firebase console

```
firebase init
```

# To start the server

```
npm run server
```

# To create react project using npm & yarn

```
npm create-react-app folder-name
```

```
yarn create react-app folder-name
```

# To start the client

```
yarn start
```
